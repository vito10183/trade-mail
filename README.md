# 📊 統一證券交易郵件自動化處理 (精簡優化版)

自動從 Gmail 擷取統一證券的正式「交易明細表」郵件，並將資料精準整理至 Google Sheets。

## ✨ 功能特色

- 📧 **自動解析郵件** - 專注處理「交易明細表」郵件格式，確保手續費、交易稅 100% 精確。
- 📈 **交易紀錄管理** - 擷取買賣來源（支援 APP、智慧單、行動語音等）、價格、數量、淨收付等完整交易資料。
- 📦 **庫存明細追蹤** - 記錄每日最新持股狀態，包含現股、融資、融券與現股庫存市值。
- 🔄 **智慧去重與優化** - 自動識別重複資料。採用優化演算法，提升大型工作表的清理與排序效率。
- 🔤 **代碼自動格式化** - 自動修正股票代碼（如 `50` 轉 `0050`、特別支援 `9816` 轉 `009816`），避免前導零遺失。
- ⏰ **定時執行** - 支援每 12 小時自動觸發處理，無痛自動記帳。

## 🚀 使用方式

### 1. 設定 Google Apps Script
1. 開啟 [Google Apps Script](https://script.google.com/)。
2. 將精簡優化後的 `Code.gs` 程式碼複製並貼上。
3. 綁定你的目標 Google Sheets 試算表。

### 2. 授權權限
首次執行需授權以下權限：
- Gmail 郵件讀取與標籤管理
- Google Sheets 試算表讀寫

### 3. 執行與排程
- **手動執行**：執行 `processAllEmails()` 函數即可立即讀取信箱、寫入並清理工作表。
- **自動排程**：執行一次 `createTimeTrigger()`，即可建立**每 12 小時**自動執行的觸發器。

## 📬 支援的郵件格式
- **寄件者**: `service@mailagent.pscnet.com.tw`
- **主要主旨**: `交易明細表`（不包含即時成交回報，避免手續費為 0 的暫時資料干擾）

## 🏷️ Gmail 標籤
處理完成的郵件會自動標記，避免重複讀取：
- `已處理-交易明細表`
